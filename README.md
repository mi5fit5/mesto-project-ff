## Описание проекта
Проектная работа Mesto — интерактивная веб-страница, куда пользователи могут публиковать фотографии интересных мест, а другие в свою очередь ставить им лайки. Это фронтенд-проект, реализованный в рамках учебного курса от Яндекс.Практикума. Все данные синхронизируются с сервером через REST API.

В проекте реализован следующий функционал:
* Редактирование профиля пользователя (имя и информацию о себе);
* Добавление новых карточек с фотографиями и названиями мест;
* Удаление собственных карточек;
* Постановка и снятие лайков на карточках;
* Просмотр фотографий в полноэкранном режиме;
* Обновление аватара пользователя.

## Технологии
* HTML5 (семантическая вёрстка)
* CSS3 (Flexbox, Grid, медиазапросы, плавные переходы)
* БЭМ (Flat/Nested)
* JavaScript (ES6+, DOM API, Promise, Fetch API)
* Webpack (Babel, PostCSS)
* Git и GitHub Pages
* REST API
